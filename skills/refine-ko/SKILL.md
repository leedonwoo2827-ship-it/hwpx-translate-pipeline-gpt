# refine-ko — 한국어 번역 자연스러움 교정(윤문) 패스

파이프라인 **03-refine** 단계의 교정 프롬프트/규칙. 입력 = 영문 원문(01-extract의
`content.en.md`) + 초벌 한국어(02-translate의 `content.ko.md`). 출력 = 자연스럽게 다듬은
한국어(03-refine의 `content.ko.md`). **단일 ‘최선의 교정본’**을 생성한다(다중안 아님).
지금은 Claude가, 향후 로컬 LLM이 동일 계약으로 수행한다.

## 핵심 원칙 (사용자 지정)

1. **원본 유지**: 새로운 정보를 추가하거나 핵심 주장을 변경하지 마시오. 사실·수치·인용·
   논지·구조를 그대로 보존한다(요약·삭제·의역으로 인한 의미 변화 금지).
2. **어조 맞춤**: 대상 독자(국내 대학·정책 실무자)가 이해하기 쉽도록 **친절하고 전문적인
   톤**으로 윤문한다. 학술적 정확성은 유지하되 딱딱한 직역투를 자연스러운 한국어 문어로.
3. **포맷팅**: 지정된 마크다운 구조/문서 포맷 규칙을 엄격히 따른다. 아래 “포맷 규칙” 참조.

## 교정 체크리스트 (직역투 → 자연스러운 한국어)

- **직역 구문 해체·재구성**: 영어 어순을 그대로 옮긴 문장을 한국어 논리로 재배열.
  - “…에 도착하며, 그것은 단지 또 하나의 …가 아니다” → “…에 도착하는데, 이는 단지 또 하나의
    …가 아니다” (또는 두 문장으로 분리).
  - “…에까지 손을 뻗기에 다르다” → “…에까지 손을 뻗는다는 점에서 다르다”.
  - “사례들을 닫는다(closes the cases)” → “사례 소개를 마무리한다”.
  - “요하네스버그 대학교에서 쓴다(writes from)” → “요하네스버그 대학교에서 이 글을 쓴다/집필한다”.
  - “…을 직접 작동시킨다(operationalizes)” → “…을 직접 구체화한다/실행에 옮긴다”.
- **외래 인명·용어 판독성**: 첫 등장 시 원어 병기, 어색한 음역·역어 순화.
  - “이 가세트의 『대학의 사명』” → “오르테가 이 가세트(Ortega y Gasset)의 『대학의 사명』”.
  - “인구학적 배당” → “인구 배당(demographic dividend)” 또는 “인구 보너스”.
  - “이중용도” → “이중용도(민·군 겸용, dual-use)”.
- **반복 표현 다양화**: “그의 장은…”, “~라고 명명한다” 등 기계적 반복을 문맥에 맞게 변주.
- **문말·연결 자연화**: 어색하게 끊긴 문말(“…시작한다.”, “…첨예하다.”)을 앞뒤와 매끄럽게
  연결하거나 능동·구체 표현으로. 단, 문장을 임의로 합치며 의미를 흐리지 말 것.
- **조사·띄어쓰기·맞춤법**: 굵게(`**`) 경계에서 조사/공백 유지, 국립국어원 규범 준수.

## 포맷 규칙 (문서공방 hwpx 마크다운 계약)

- `# 제목1`(장 제목) / `## 제목2`(소제목) / 첫 줄 `**저자**`. 헤딩 텍스트에 마크다운 기호 금지.
- 문단=본문, `- `=동그라미(○) 1단계, `  - `=마이너스(−) 2단계, `> `=인용, `**강조**`.
- 도표: `![캡션](figures/fig-01.png)` — **경로·파일명 절대 변경 금지**(build가 BinData 삽입).
- 장 말미 `## 분석 (내부 검토용)` 섹션(요약/핵심 논점/시사점)은 유지·동일 윤문.
- 참고문헌 축약 표기 유지. 원문에 없는 절/도표/링크를 새로 만들지 말 것.

## 절차 (한 장)

1. `content.en.md`(원문)와 `content.ko.md`(초벌)를 문단 단위로 대조.
2. 위 체크리스트로 각 문단을 윤문 — 의미·구조·포맷 보존.
3. 결과를 03-refine의 `content.ko.md`로 저장.
4. 검토: 웹 뷰어에서 **초벌(02) 대비 diff**로 변경점 확인, 전문가가 04-review에 확정.

## 로컬 LLM 이관 시

동일 입력/출력 계약(EN md + 02 KO md → 03 KO md)과 위 프롬프트를 시스템 프롬프트로 사용.
장 단위로 스트리밍 처리하고, diff로 회귀(의미 변형·포맷 훼손)를 점검한다.
