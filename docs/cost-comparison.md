# 번역 비용 비교 — ChatGPT(codex) vs Claude vs DeepSeek

게시판 설명용. **기준 문서**: `SQL-to-Dashboard_Transcript_Part01.pdf`
- 39페이지, 영문 약 122,000자(약 24,000단어)
- 추정 토큰: **입력 ~31K**, 한국어 완역 **출력 ~60K** (번역 1건 기준)
- 윤문(03)은 원문+초벌을 다시 넣어 **입력 ~90K / 출력 ~60K** → 번역의 약 1.6배

> ⚠️ 토큰·가격은 추정치입니다. 정확히는 각 API의 토크나이저/`count_tokens`로 재측정하세요. 단가는 수시 변동.

## 과금 방식 3가지 (핵심)

| 엔진 | 과금 방식 | 이 프로그램에서 |
|---|---|---|
| **ChatGPT (codex OAuth)** | **구독 정액**(ChatGPT Plus/Pro 또는 워크스페이스 크레딧)의 **사용량 한도** 소모. 건당 종량 과금 아님 | 현재 방식. 한도 초과 시 "out of credits" |
| **Claude Code** | (a) **구독 정액**(Claude Pro/Max) 또는 (b) **API 종량**(토큰당) | 미연동 |
| **DeepSeek** | **API 종량**(토큰당, 매우 저렴) | 미연동 |

즉 **codex와 Claude(구독)는 "정액 안에서 사용량 소모"**, **DeepSeek·Claude(API)는 "쓴 만큼 토큰 과금"**.

## 이 문서 1건(39p) 번역 비용 — API 종량 환산

정확 비교를 위해 모두 **토큰당 API 단가**로 환산(입력 31K + 출력 60K):

| 엔진 / 모델 | 입력 $/1M | 출력 $/1M | 이 문서 1건(번역) | 원화(≈1,450원/$) |
|---|---|---|---|---|
| **ChatGPT codex** (mini급 환산*) | ~0.15 | ~0.60 | **~$0.04** | **~60원** |
| **Claude Haiku 4.5** | 1.00 | 5.00 | ~$0.33 | ~480원 |
| **Claude Sonnet 5** (정가) | 3.00 | 15.00 | ~$0.99 | ~1,440원 |
| **Claude Sonnet 5** (인트로**) | 2.00 | 10.00 | ~$0.66 | ~960원 |
| **Claude Opus 4.8** | 5.00 | 25.00 | ~$1.66 | ~2,400원 |

\* codex는 실제로 API 종량이 아니라 ChatGPT 구독 할당량을 씁니다. 위 $는 "동급 mini API로 환산 시" 참고치.
\** Sonnet 5 인트로 요금은 2026-08-31까지.

**윤문까지 하면**(입력 ~90K/출력 ~60K) 각 모델 비용이 대략 위의 **1.5~1.7배** 추가됩니다.

## 요약 (게시판용)

- **가장 저렴/현행**: **ChatGPT(codex) — 구독 정액**. 건당 추가 과금 없이 ChatGPT Plus/Pro 한도 안에서 사용. 대량이면 상위 구독.
- **DeepSeek**: API 종량 중 가장 쌈(문서 1건 ~100~200원). 별도 API 키·결제 필요.
- **Claude**: 품질 최상급이나 종량 단가 높음(Opus 문서 1건 ~2,400원). Claude Code를 **Max 구독**으로 쓰면 정액.
- **윤문(03)은 토큰을 1.6배 더 씁니다 → 기본은 번역(02)까지만, 윤문은 특히 어색한 장에만.**
